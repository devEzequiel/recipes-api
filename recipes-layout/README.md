## Running the app


To run the react app in your local environment, first **create a new terminal instance**. 

In Visual Studio Code a new terminal instance is created with
<br />

```ctrl + shift + ` ```

Then, go to 

`$ cd recipes-layout`

Done that

`$ npm install`

`$ npm start`

Ready, now you can use the app.
